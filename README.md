# gl3DGE
OpenGL 3D Game Engine

gl3DGE is an easy-to-use but flexible game engine for creating 3D games in Java.
gl3DGE uses OpenGL for rendering through LWJGL (separate library required).

********

gl3DGE is supports:
* OBJ loading
* Texture maps
* Specularity maps
* Emission maps
* Smooth shading
* Per-fragment Phong shading
* 3D transformations
